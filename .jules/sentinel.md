## 2024-03-18 - [Path Traversal in Workspace Resolution]
**Vulnerability:** The `_resolve_path` function in `src/bub/builtin/tools.py` allowed absolute paths to entirely bypass the workspace confinement checks (`if path.is_absolute(): return path`). Additionally, relative paths using `../` could easily escape the base directory because `(workspace_path / path).resolve()` was not explicitly validated against `workspace_path.is_relative_to()`.
**Learning:** Never assume that simple path joining (`/`) combined with `Path.resolve()` inherently sandboxes a path. If the RHS of the join is absolute, Python's `Path` implementation effectively ignores the LHS. If the RHS contains sufficient `../` segments, the resolved absolute path will step outside the intended root.
**Prevention:** Always compute the fully resolved absolute path (`target = (root / raw).resolve()`) and rigorously verify `target.is_relative_to(root)`. If supporting absolute paths supplied by an agent, strip leading slashes before joining to effectively re-base the path into the sandbox (`target = (root / str(raw).lstrip('/')).resolve()`).
