## 2024-05-18 - Path Traversal in File Tools
**Vulnerability:** The `_resolve_path` function in `src/bub/builtin/tools.py` allowed absolute paths to bypass workspace restriction checks, and did not properly validate if relative paths (e.g. `../`) broke out of the workspace directory. This allowed arbitrary file read/write on the host system.
**Learning:** Returning `Path` directly if `.is_absolute()` is true bypasses all subsequent relative checking logic. Additionally, `Path.resolve()` needs to be compared against the workspace path to ensure path containment, particularly because `.expanduser()` or multiple parent directory traversals (`../`) can escape intended boundaries.
**Prevention:** Always sandbox path operations. If the path is absolute, strip its root anchor to make it relative to the workspace. Use `resolved_path = (workspace_path / path).resolve()` and explicitly check `resolved_path.is_relative_to(workspace_path)`. Raise a `PermissionError` if it escapes the workspace bounds.
